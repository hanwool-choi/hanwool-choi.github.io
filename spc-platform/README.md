# SPC 오퍼레이션 플랫폼 — Web 프로토타입 (v2.2 기능정의 기반)

`SPC 오퍼레이션 플랫폼 기능도출_v2.2` IA·권한 매트릭스·맵레이어 정의를 그대로 구현한 클릭 가능한 Web 프로토타입입니다.
외부 라이브러리 없이 순수 HTML/CSS/JS로 작성되어 오프라인에서도 동작합니다.

## 실행 방법

**방법 1 — 그냥 열기**: `index.html` 더블클릭 (크롬 권장)

**방법 2 — 로컬 서버** (권장):
```
node server.js
→ http://localhost:5173
```

## 둘러보기 가이드

| 기능 | 위치 |
|---|---|
| **역할 전환** (권한 매트릭스 시연) | 우측 상단 아바타 → SPC Admin / 법인 관리자 / 개인 농민 / 대행 오퍼레이터. 역할에 따라 메뉴·버튼·데이터 범위가 바뀝니다 (예: Admin은 재무 접근 불가, 오퍼레이터는 4개 메뉴만) |
| **통합 모니터링** (Single Map, Multi-Layer) | 사이드바 2번. 우측 레이어 독에서 10개 레이어 on/off, 정밀농업 레이어 켜면 하단 타임라인(토양→생육→처방→수확) 등장. 장비 마커가 실시간으로 이동하며 커버리지가 차오릅니다 |
| **딥링크 규약** | 장비/작업/정밀농업 화면의 [맵] 버튼 → 통합 맵이 해당 레이어·필지 프리셋으로 열림 |
| **A-Motion 관제** | 맵에서 HX1400 마커 클릭 → 관제 모드, 또는 작업관리 > A-Motion |
| **농작업 대행** (현행 플랫폼 화면 반영) | 작업관리 > 농작업 대행: 신청접수(체크 선택 시 하단 액션바) → 대행현황(팀별 진행률+필지 지도) → 정산(영농일지·감액 사유·정산서) |
| **AI Agent** | 우하단 플로팅 버튼. 추천 질문 칩: 장비 위치 조회 → 맵 딥링크, 자연어 통계 → 차트 응답, to-do 브리핑, 매뉴얼 검색 |
| **커맨드 팔레트** | `Ctrl+K` — 91개 기능·필지·장비 통합 검색 |
| **경계 편집기** | 농장관리 > 경작지 경계 > 경계 편집기 열기 (현행 '농장 경계 편집' 모달 스타일) |

## 구조

```
index.html          앱 셸
css/main.css        디자인 토큰 + 전체 스타일
js/data.js          목데이터 (v2.2 IA 메뉴·권한 매트릭스·필지·장비·작업·정밀농업)
js/map.js           통합 맵 엔진 (SVG, 레이어 시스템, 실시간 애니메이션, 타임라인)
js/views.js         대시보드·농장·장비·작업 화면
js/views2.js        자재·정밀농업·재무·통계·시스템 화면
js/charts.js        차트 헬퍼 (bar/line/donut/spark)
js/app.js           라우터·역할 전환·AI Agent·커맨드 팔레트
```

※ App 버전은 본 Web 기능 확정 후 별도 구현 예정 (권한·채널 매트릭스의 App 컬럼 참조)
